# Security Policy

## Supported Versions

This repository is a research prototype developed for academic purposes.

| Version | Supported |
|---|---|
| v0.5-vkr | yes |

## Reporting a Vulnerability

If you find a security issue, please use GitHub private vulnerability reporting when available.

Please do not open public issues for potential vulnerabilities before they are reviewed.

## Scope

This project does not process real user data, credentials, production secrets, or live network traffic by default. The current implementation is intended for simulation and research use.
